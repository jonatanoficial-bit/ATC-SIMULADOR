# Fase 28 — Academia ATC e Certificações

A Fase 28 adiciona uma camada jogável de treinamento e certificação.

## Sistemas adicionados

- Academia ATC;
- trilhas de treinamento;
- missões guiadas;
- provas de certificação;
- avaliação por rubrica;
- recomendações de estudo;
- progressão didática;
- painel `ATC ACADEMY`.

## Trilhas

- Torre Básica.
- Controle de Solo.
- Aproximação Radar.
- Procedimentos IFR.
- Emergências e Multiagência.
- Network Flow e Slots.

## Impacto no gameplay

Ao finalizar o turno, a build avalia:

- missão ativa;
- objetivos cumpridos;
- segurança;
- procedimento correto;
- eficiência;
- comunicação/fluxo;
- aprovação/reprovação;
- próxima missão sugerida.
