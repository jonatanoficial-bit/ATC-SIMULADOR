# Guia de feedback público — Fase 27

## Categorias

- Gameplay / realismo
- Mobile / toque / rolagem
- Performance / travamento
- Procedimentos ATC
- Visual / HUD
- Áudio / alertas

## Severidades de bug

- BLOCKER: impede jogar.
- CRITICAL: quebra função essencial.
- MAJOR: prejudica experiência, mas não bloqueia totalmente.
- MINOR: texto, visual ou ajuste leve.

## Critério de ação

- BLOCKER/CRITICAL: hotfix antes de divulgar.
- MAJOR: patch curto.
- MINOR: polish patch.
