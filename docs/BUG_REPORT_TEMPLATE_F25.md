# Modelo de Bug Report — Fase 25

## Severidade

- BLOCKER: tela branca, boot quebrado, jogo impossível.
- CRITICAL: comando essencial quebra, save corrompe, turno trava.
- MAJOR: layout inacessível, botão fora da tela, scroll falho.
- MINOR: texto, estética, micro ajuste.
