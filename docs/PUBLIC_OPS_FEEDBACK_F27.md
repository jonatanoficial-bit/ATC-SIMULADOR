# Fase 27 — Public Ops, Feedback e Hotfix Planner

A Fase 27 adiciona uma camada de sustentação pública offline/local para o Skyward Control.

## Sistemas adicionados

- feedback local do jogador;
- telemetria offline;
- caixa de bugs;
- triagem por severidade;
- plano automático de hotfix;
- exportação de dossiê JSON;
- painel `PUBLIC OPS`;
- status de estabilidade pública.

## Observação

Nenhum dado é enviado para servidor externo. A telemetria é local/offline e pode ser exportada manualmente.
