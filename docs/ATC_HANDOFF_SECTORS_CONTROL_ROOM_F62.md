# ATC Handoff, Setores e Sala de Controle — Fase 62

## Setores

- GND — Solo.
- TWR — Torre.
- DEP — Partida.
- APP — Aproximação.
- CTR — Centro.
- FMP — Fluxo.
- EMG — Mesa de emergência.

## Tipos de transferência

- Solo → Torre.
- Torre → Partida.
- Aproximação → Torre.
- Centro → Aproximação.
- Torre → Solo.
- Fluxo → Torre.
- Emergência → todos.

## Regras de coordenação

- Troca de frequência.
- Identificação radar.
- Liberação de pista.
- Aproximação perdida.
- Liberação de partida.
- Transferência emergencial.
- Desvio por clima.
