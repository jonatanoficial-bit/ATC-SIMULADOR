# Fase 33 — Crisis Command Center

A Fase 33 adiciona um centro de comando para operações irregulares e crises sistêmicas.

## Sistemas adicionados

- Ground stop regional;
- falha elétrica parcial;
- incidente cyber operacional;
- greve/equipe reduzida;
- célula severa;
- alerta de segurança no terminal;
- ações de comando;
- recovery plan;
- stakeholders;
- painel `CRISIS CMD`.

## Impacto no gameplay

Ao finalizar o turno, a build calcula:

- pressão sistêmica;
- crise ativa;
- estágio de recuperação;
- ações de mitigação;
- score de resiliência;
- status de comando.
